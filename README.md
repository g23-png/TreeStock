# Tree-Stock

## Objetivo

Desarrollar un sistema de inventario utilizando un Árbol Binario de Búsqueda (ABB) en Java para registrar, buscar y listar productos de manera ordenada.

## Estructura del Proyecto

* Producto.java
* ArbolInventario.java
* Main.java

## Funcionalidades

1. Registrar productos.
2. Mostrar inventario ordenado por ID.
3. Buscar productos por ID.
4. Salir del sistema.

## Ejecución

Compilar:

javac *.java

Ejecutar:

java Main
